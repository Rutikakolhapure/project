## 🚀 Why This Project Matters

Most agri-tech solutions solve **one problem**.  
This system solves **four critical problems simultaneously**:

✔ Soil Classification  
✔ Nutrient Estimation (NPK + pH)  
✔ Plant Disease Detection  
✔ Weather-Aware Crop Recommendation  

➡️ Built as a **production-style full-stack AI system**, not just a model.

---

## 🧠 What Makes This Different

- End-to-end pipeline: **Image → AI → Recommendation**
- Combines **Computer Vision + Structured Data + APIs**
- Real-world usable (not just academic)
- Designed for **low-resource farmers (mobile-first)**

## 🤖 Models & Performance

### 🌍 Soil Classification
- CNN-based model  
- Input: 150×150 images  
- Accuracy: ~88%  

### 🌿 Disease Detection
- Transfer Learning (ResNet / EfficientNet)  
- Input: 224×224 images  
- Accuracy: ~88%

## Getting Started

### 1. Clone Repository
```bash
git clone https://github.com/your-username/agro-optics.git
cd agro-optics
```

## 2. Backend Setup
pip install -r requirements.txt
python app.py

## 3. Frontend Setup
cd frontend
npm install
npm start
